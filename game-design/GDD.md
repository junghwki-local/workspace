# 게임 기획서 (GDD)
> 머지 RPG 던전 탐험 게임 — v1.0

---

## 1. 게임 개요

| 항목 | 내용 |
|------|------|
| 장르 | 머지 + 액션 RPG |
| 플랫폼 | 모바일 (iOS / Android) |
| 엔진 | Unity |
| 핵심 경험 | 장비를 합성해 강화하고, 파티를 꾸려 던전을 탐험하는 즐거움 |
| 세션 시간 | 10~30분 |

---

## 2. 핵심 게임 루프

```
던전 탐험 → 골드 + 장비 드롭 획득
        ↓
대장간에서 골드로 무기/방어구 생산
        ↓
머지 인벤토리에서 장비 합성 (등급 강화)
        ↓
파티원에게 장비 장착 + 장신구로 속성 세팅
        ↓
더 높은 던전 탐험
```

---

## 3. 머지 시스템

### 기본 규칙
- **같은 등급** 2개 합성 → 한 단계 위 등급 생성
- 등급이 다르면 합성 불가

### 장비별 합성 규칙
| 장비 | 합성 조건 |
|------|---------|
| 무기 / 방어구 | 같은 등급이면 합성 가능 |
| 장신구 (반지/목걸이/귀고리) | 같은 등급 + **같은 속성**이어야 합성 가능 |

### 등급 체계
```
1등급(일반) → 2등급(고급) → 3등급(희귀) → 4등급(영웅) → 5등급(전설) → ...
```

### 인벤토리
- 머지 보드: **10 x 10 그리드 (100칸)**
- 창고: 별도 보관 공간, 창고에서도 바로 합성 가능

---

## 4. 장비 시스템

### 장비 슬롯 (캐릭터당 5슬롯)

| 슬롯 | 역할 |
|------|------|
| 무기 | 공격력 + 스킬 부여 (무속성) |
| 방어구 | 방어력 (무속성) |
| 반지 | **무기에 속성 부여** |
| 목걸이 | **방어구에 속성 부여** |
| 귀고리 | 버프 효과 (힐, 공속, 쿨타임 감소 등) |

### 속성 시스템
- 무기/방어구는 자체 속성 없음 (무속성)
- 반지/목걸이 장착 시 해당 속성이 무기/방어구에 적용
- 장신구 없으면 무속성 유지 (패널티 없음)

| 속성 | 색상 |
|------|------|
| 불 | 빨강 |
| 물 | 파랑 |
| 바람 | 초록 |
| 땅 | 황토 |
| 빛 | 노랑 |
| 어둠 | 보라 |

### 장비 획득처
| 장비 | 획득처 |
|------|------|
| 무기 / 방어구 | 대장간 생산 + 던전 드롭 |
| 장신구 | 던전 드롭으로만 획득 |

---

## 5. 대장간 시스템

- 골드 소모 → 무기/방어구 **즉시 생산**
- 생산 결과: 등급 랜덤 (무속성)
- 대장간 레벨 업그레이드 (골드 소모) → 고등급 장비 출현 확률 증가

| 골드 사용처 | 내용 |
|---------|------|
| 장비 생산 | 무기/방어구 즉시 생산 |
| 대장간 업그레이드 | 고등급 확률 증가 |

---

## 6. 캐릭터 시스템

### 파티 구성
- **주인공 1명**: 플레이어 직접 조작
- **파티원 2명**: 주인공 뒤를 따라다니며 자동 전투
- 전원 5슬롯 장비 독립 운용

### 직업 (4종)

| 직업 | 포지션 | 특성 |
|------|------|------|
| 전사 | 근거리 | 높은 방어력, 어그로 유도, 방어구 효과 증가 |
| 마법사 | 원거리 | 광역 스킬, 장신구 쿨타임 감소, 낮은 방어력 |
| 힐러 | 원거리 | 아군 회복, 귀고리 버프 효과 증가, 부활 쿨타임 감소 |
| 궁수 | 원거리 | 빠른 공격속도, 무기 속성 데미지 보너스, 높은 기동성 |

### 사망 처리
- 사망 후 일정 시간이 지나면 자동 부활
- 부활 아이템 사용 시 즉시 부활

---

## 7. 전투 시스템

### 기본 조작
- **이동**: 화면 왼쪽 조이스틱
- **공격**: 자동 (적 근접 시 자동 공격 — Vampire Survivors 방식)
- **스킬**: 주인공은 수동 발동 / 파티원은 자동 발동

### 일반 전투
- 자동으로 진행, 장비 스펙이 충분하면 조작 없이 클리어

### 보스 전투
- 보스 고유 패턴 존재
- 스킬 타이밍 조절 필요
- 보스 속성 약점에 맞는 장신구 세팅이 승패를 결정
- 고스펙이어도 전략 없으면 클리어 어렵게 설계

---

## 8. 던전 시스템

### 던전 종류
| 타입 | 설명 |
|------|------|
| 일반 던전 | 자동 전투, 골드 + 장비 드롭 |
| 보스 던전 | 패턴 공략 필요, 고등급 드롭 |
| 특수 던전 | 추후 추가 (이벤트, 속성 특화 등) |

### 맵 구조
- 구조 고정 (예: 3구역, 구역당 방 5개)
- 방 내부 몬스터 배치 + 보상은 랜덤
- 보스는 고정

### 속성 약점
- 보스마다 약점 속성 1~2개 표시
- 약점 속성 장착 시 데미지 보너스
- 반대 속성 장착 시 데미지 패널티

### 보상
- 골드
- 장비 드롭 (등급별 확률 차등, 고레벨 던전일수록 고등급 확률 증가)
- 장신구는 던전 드롭으로만 획득 가능
